🎨 Generador de Iconos PWA - KS Agente

Haz clic en los botones para descargar los iconos en formato PNG:

192x192


512x512


📝 Instrucciones:

  1. Haz clic en cada botón para descargar los iconos
  2. Guarda los archivos como icon-192.png e icon-512.png
  3. Colócalos en la carpeta public/ del proyecto
  4. Haz build del frontend: npm run build
  5. Despliega a CloudFront
